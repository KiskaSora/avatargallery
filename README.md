# 🖼 Avatar Gallery

Галерея аватарок прямо внутри SillyTavern. На аватарках появляется значок 🖼 —
клик открывает набор картинок для **этой** аватарки. Выбираешь картинку → она
**реально становится** аватаркой персоны или персонажа в таверне и показывается
везде (в чате, в списках, в панели).

Картинки хранятся локально в браузере (IndexedDB), поэтому `settings.json` не
раздувается. Дизайн автоматически подстраивается под текущую тему ST.

---

## Возможности

- **Своя галерея у каждой аватарки** — отдельный набор картинок для каждой персоны
  и каждого персонажа.
- **Реальная замена аватарки** — выбранная картинка через API SillyTavern
  заменяет файл аватарки. Меняется везде, а не только визуально на один сеанс.
- **Значки во многих местах:**
  - список персон (Управление персонами);
  - аватар персонажа в панели редактирования;
  - аватарки в сообщениях чата *(по наведению, можно выключить)*;
  - список персонажей *(по наведению, можно выключить)*.
- **Менеджер аватарок** в меню-палочке — все сохранённые галереи в одном окне:
  применить, удалить картинку, удалить целую галерею.
- **Горизонтальная лента миниатюр** — листается влево/вправо; стрелки ‹ › и клики
  переключают большое превью по всем картинкам.
- **Ничего не теряется** — при первом открытии текущая аватарка автоматически
  кладётся в галерею первой, так что всегда можно вернуть.
- **Тема ST** — цвета, фон, рамки берутся из переменных темы (`--SmartTheme*`).

---

## Установка

1. Папку `avatar-gallery` положить в
   `SillyTavern/public/scripts/extensions/third-party/`.
2. Перезагрузить страницу SillyTavern (Ctrl+Shift+R).
3. Готово — расширение включается само.

---

## Как пользоваться

1. Наведи курсор на аватарку (персоны, персонажа, в сообщении или в списке) и нажми
   значок **🖼** в углу.
2. **«Загрузить картинки»** — добавь сколько угодно вариантов.
3. Пролистай ленту, выбери нужную миниатюру.
4. Нажми **«Сделать аватаркой»** — картинка станет реальной аватаркой.
   Текущая помечается галочкой ✓.

### Менеджер аватарок
Меню-палочка (🪄) → **«Менеджер аватарок»**:
- клик по миниатюре — применить её как аватарку;
- ✕ на миниатюре — удалить картинку;
- 🗑 у названия — удалить всю галерею этой аватарки;
- клик по названию — открыть полную галерею.

## Настройки

**Настройки → Расширения → «Галерея аватарок»:**

| Опция | Что делает |
|---|---|
| Включить галерею | Общий выключатель (значки + пункт меню-палочки). |
| Значки в сообщениях чата | Вкл/выкл значков на аватарках в чате. |
| Значки в списке персонажей | Вкл/выкл значков в списке персонажей. |

---

## Как это работает

- **Персона** → `POST /api/avatars/upload` с `overwrite_name` (перезапись файла
  персоны, имя и описание не трогаются).
- **Персонаж** → `POST /api/characters/edit-avatar` с `avatar_url` (замена аватара
  карточки).

После замены расширение сбрасывает кэш миниатюр, чтобы новая аватарка показалась
сразу во всех местах.

## Хранение

- Картинки галереи → IndexedDB, база `avatar_gallery_db`, хранилище `galleries`.
  Ключ записи: `persona:<файл>` или `char:<файл>`.
- В `settings.json` хранятся только флаги вкл/выкл.

---

## Если что-то не так

- **Не появился значок** — обнови страницу (Ctrl+Shift+R); проверь, что галерея
  включена в настройках.
- **Не применяется аватарка** — открой консоль (F12) и посмотри ошибки с пометкой
  `[AvatarGallery]`; обычно это ответ API таверны.
- **Картинки пропали** — они в IndexedDB конкретного браузера/профиля; очистка
  данных сайта их удалит.

---

## Совместимость

Работает независимо. Никак не связано с расширениями генерации картинок —
меняет именно реальные аватарки SillyTavern.
